# E-commerce Project

## Overview
This is a **full-stack e-commerce web application** built with **React, Bootstrap, and Django**.  
Users can browse products, search, view deals, add items to the cart, and shop seamlessly online.  

---

## Features
- Responsive **frontend** using React and Bootstrap  
- Dynamic **product listing** fetched from Django backend  
- **Search products** by name  
- **Add to Cart** functionality  
- **Carousel** for promotions and sales  
- **Shop by Category** section  
- **Deal of the Day** section  
- Free delivery and easy returns notifications (simulated)  

---

## Technologies Used
- **Frontend:** React, Bootstrap, JavaScript, HTML, CSS  
- **Backend:** Python, Django, Django REST Framework  
- **Database:** SQLite (can be replaced with PostgreSQL)  
- **Version Control:** Git, GitHub
